Basic Loot Table

Games often have a random chance of dropping rewards when the player kills a monster. Various methods exist to determine what items drop. One simple method is to create an array of items and assign them each a chance of dropping. This chance is called the 'weight' or 'drop weight'. When it is time to drop loot a random number is generated and compared with the drop weights of each item in the array. Whichever item matches the number is dropped. The forumla is: Add up the weights for all items in the array. Generate a random number between 1 and the total. Iterate through each item and compare the number to the item's weight. If it is equal or less than the weight, this item is dropped. If not, move on to the next item. This website demonstrates a basic loot drop table in action.
